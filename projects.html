A Lead Developer to Stick Flick.
This game is a remodelling of the old flash Defend Your Tower game designed for mobile android use. You play from the perspective of your castle, and you flick StuckDude enemies away from your castle until they take enough damage to die. There are various types of StickDudes and power-ups to add a bit of variance to gameplay. This is developed using Java and the LibGDX libraries.

A Lead Developer to Kuroko no Databasuke.
This project developed a two person chat that used only images for communication. Only the most recent image sent is shown in the chat, and the chat display is updated as soon as the database is updated with a new sent/received image. It was developed using a combination of HTML, CSS, PHP, JavaScript, and PostgreSQL.

A contributor to Security Cheatsheets.
This project expands upon the default set of cheatsheets, and the purpose of these cheatsheets are to aid penetration testers/CTF participants/security enthusiasts in remembering commands that are useful, but not frequently used. The "sheets" themselves are written in plain text, and run using the cheatsheet setup from the main library.
